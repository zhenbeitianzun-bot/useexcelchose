# 评审人员项目库随机抽取系统

一个轻量级的跨平台工具，用于从评审人员库中随机抽取评审人员，并导出符合规范的Excel记录表。

## 功能特点

- **随机抽取**：支持按部门抽取评审人员，确保公平公正
- **部门管理**：可设置申请部门、关联部门和排除部门
- **自动生成Excel**：导出符合规范的评审人员项目库随机抽取记录表
- **响应式设计**：同时支持电脑端和手机端使用
- **离线使用**：无需网络连接，随时随地可用
- **自动字体调整**：根据抽取人员数量自动调整Excel中的字体大小
- **抽取情况明细**：在Excel中自动生成按部门分组的抽取人员明细

## 技术特性

- 基于纯HTML5+CSS3+JavaScript开发，无需安装额外软件
- 使用XLSX.js和ExcelJS库处理Excel文件
- 响应式设计，适配各种屏幕尺寸
- 本地文件存储，保护数据隐私

## 文件说明

- `index.html`：系统主文件，包含所有功能逻辑
- `xlsx.full.min.js`：Excel文件处理库
- `exceljs.min.js`：Excel文件生成库
- `评审人员项目库随机抽取记录表.xls`：Excel模板文件
- `README.md`：使用说明文档

## 如何使用

### 电脑端使用

#### 方法1：直接打开（推荐）
1. 双击 `index.html` 文件，系统将在默认浏览器中打开
2. 开始使用系统功能

#### 方法2：使用本地服务器（推荐）
1. 确保安装了Python
2. 在文件所在目录打开命令提示符
3. 运行命令：`python -m http.server 8000`
4. 在浏览器中访问：`http://localhost:8000`

### 手机端使用

#### 方法1：通过电脑共享
1. 在电脑上使用本地服务器方式启动系统
2. 确保手机和电脑连接在同一WiFi网络
3. 查看电脑的IP地址（Windows：`ipconfig`，Mac：`ifconfig`）
4. 在手机浏览器中输入：`http://电脑IP:8000`

#### 方法2：直接传输文件
1. 将所有文件传输到手机中（可通过微信、QQ等工具）
2. 使用手机文件管理器找到 `index.html` 文件
3. 点击打开，选择使用浏览器打开

## 操作步骤

### 1. 上传人员数据库
1. 点击"选择文件"按钮，选择包含评审人员信息的Excel文件
2. 点击"处理文件"按钮，系统将解析文件内容

### 2. 填写抽取信息
1. 输入采购项目名称
2. 输入申请人姓名
3. 输入监督人员姓名
4. 选择申请部门
5. 选择关联部门（可添加多个）
6. 选择排除部门（可添加多个）

### 3. 进行人员抽取
1. 输入需要抽取的人员数量
2. 点击"开始抽取"按钮
3. 系统将自动抽取符合条件的人员

### 4. 导出Excel记录
1. 点击"导出Excel"按钮
2. 系统将生成并下载Excel文件
3. 打开下载的Excel文件，检查内容是否正确

## 使用注意事项

- **人员数据库格式**：Excel文件需包含"人员"和"部门"两列
- **文件大小限制**：建议单次上传的Excel文件不超过1MB
- **浏览器兼容性**：建议使用Chrome、Firefox、Safari或Edge最新版本
- **手机端操作**：在手机上操作时，建议使用横屏模式以获得更好的体验

## 常见问题及解决方法

### Q: 无法打开系统
A: 请确保所有文件在同一目录下，建议使用Chrome浏览器打开

### Q: 上传Excel文件后没有反应
A: 请检查Excel文件格式是否正确，确保包含"人员"和"部门"列

### Q: 抽取人员后导出Excel失败
A: 请确保已安装最新版本的浏览器，或尝试刷新页面后重新抽取

### Q: 在手机上使用时界面显示不全
A: 请尝试使用横屏模式，或缩小浏览器页面比例
